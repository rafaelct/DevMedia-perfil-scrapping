# DevMedia-perfil-scrapping
Script em Python que faz WebScrapping de um perfil da DevMedia e mostra estatística do mesmo.


Instalar:

- Python3
- pip install mechanize
- pip install pip install bs4

Uso: 
python devmedia.py <login> <senha>
  
Exemplo de execução:

<pre>
